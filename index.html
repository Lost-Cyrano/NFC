Partage d'informations par NFC

⇅

Prêt à partager

Approchez les téléphones dos à dos

Comment utiliser :

  1. Remplissez les informations à partager
  2. Cliquez sur "Activer le partage NFC"
  3. Approchez votre téléphone du téléphone client
  4. Maintenez les téléphones proches (1-2 cm)
  5. Les informations seront transférées automatiquement
Compatible avec les appareils Android supportant Web NFC (Chrome)

Comment utiliser ce lecteur NFC

  • Cliquez sur "Démarrer la lecture NFC" pour activer le lecteur
  • Approchez un tag NFC de votre appareil (à environ 1-2 cm)
  • Les données du tag s'afficheront automatiquement
  • Cliquez sur "Arrêter la lecture" pour désactiver le lecteur
Note : Cette fonctionnalité nécessite un navigateur compatible Web NFC (Chrome pour Android) et un appareil avec un lecteur NFC.
